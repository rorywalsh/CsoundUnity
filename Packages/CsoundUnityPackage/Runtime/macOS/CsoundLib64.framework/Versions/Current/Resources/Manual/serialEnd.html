serialEnd

serialEnd — Close a serial port.

Description

Plugin opcode in serial.

Close a serial port for arduino.

Syntax

 serialEnd iPort

Initialization

iPort -- port number optained from a serialBeginopcode.

See Also

serialBegin, serialWrite_i, serialWrite, serialRead, serialPrint, serialFlush.
