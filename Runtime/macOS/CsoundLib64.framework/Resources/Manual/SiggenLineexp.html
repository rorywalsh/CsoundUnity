Linear and Exponential Generators

The opcodes that generate linear or exponential curves or segments are:

  • expon

  • expcurve

  • expseg

  • expsega

  • expsegr

  • gainslider

  • jspline

  • line

  • linseg

  • linsegr

  • logcurve

  • loopseg

  • loopsegp

  • lpshold

  • lpsholdp

  • rspline

  • scale

  • transeg

  • bpf

  • bpfcos

  • linlin

  • lincos

  • xyscale
