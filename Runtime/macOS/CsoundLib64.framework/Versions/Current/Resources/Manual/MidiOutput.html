MIDI Message Output

Opcodes that produce MIDI output are:

  • mdelay

  • nrpn

  • outiat

  • outic

  • outic14

  • outipat

  • outipb

  • outipc

  • outkat

  • outkc

  • outkc14

  • outkpat

  • outkpb

  • outkpc

  • midiout
